# **App Name**: AlloyStream

## Core Features:

- Smart Hierarchy Configuration: Admin tool to define master dimensions for coils, specific cut-to-length (CTL) sizes, and finalized parts, ensuring data integrity across the manufacturing chain.
- AI Vision Tag Scanner: Leverages a vision tool to automatically extract net weight and coil dimensions from images of physical packing tags, with immediate validation against admin presets.
- Precision Mass Calculation: Calculates weight of CTL units and scrap using a density of 7.86, automatically bridging the gap between actual net weight and calculated production mass.
- Inventory Discrepancy Shield: Automatically blocks excess weight from inventory transfers and requires admin approval before releasing the surplus to the WIP stage.
- WIP Flow Tracker: Tracks materials as they transition from raw coil weight (KG) into cut pieces and finished parts (Nos) with a live visual status of work-in-progress inventory.
- Production Capture Hub: Minimalist operator interface for logging CTL production counts and part production directly against predefined master records.
- Material Recognition AI: AI-powered tool that automatically determines if a captured document refers to a raw Coil or a processed CTL and routes the log to the correct stage.

## Style Guidelines:

- Primary color: Industrial Electric Blue (#3E79ED) to evoke precision and technical expertise.
- Background: Dark Carbon Gray (#0E1114), a heavily desaturated blue-hue base for high-contrast visibility on the factory floor.
- Accent: Arctic Cyan (#24CCCC) used for primary action buttons and scanner indicators for a high-tech finish.
- Headlines: 'Space Grotesk' for its machined and techy personality; Body: 'Inter' for maximum readability in fast-paced production environments.
- Geometric line-icons with consistent stroke weights that mirror blueprints and industrial schematics.
- Task-focused dashboard for operators with oversized buttons to minimize error when used with industrial gear or on handheld devices.
- Smooth horizontal slide transitions when moving through the production hierarchy to represent the material flow pipeline.
